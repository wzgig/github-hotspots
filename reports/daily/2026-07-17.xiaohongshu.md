01｜skills
把软件工程方法拆成可组合的 Agent 技能，帮助开发者对齐需求、测试代码并改善架构

它能做什么：

- 追问需求并梳理未决分支
- 建立项目共享术语与决策记录
- 按红绿重构循环推进开发
- 诊断难复现的缺陷与性能退化
- 扫描代码库并寻找架构改进点

核心亮点：可组合的软件工程技能库。每项技能聚焦一个工程环节，可按任务组合使用；既有手动触发的流程编排，也有 Agent 按场景调用的测试、调试、建模与审查方法。

适合谁：使用 Claude Code、Codex 等编码 Agent，希望加强需求澄清、测试、调试和代码设计的开发者
使用前准备：通过 skills.sh 安装后选择所需技能与编码 Agent，并在每个仓库运行 `/setup-matt-pocock-skills`。

使用提醒：Claude Code 插件以只读托管包更新，不能直接编辑；原生 Codex 插件仍在路线图中。

许可证：MIT

为什么本期上榜：过去 24 小时净增 +2,298 Star（历史快照核验）
GitHub 搜索：mattpocock/skills

02｜hermes-agent
让 AI Agent 跨终端和聊天平台执行任务，并从历史会话中积累记忆与可复用技能

它能做什么：

- 切换模型提供方而无需修改代码
- 搜索历史会话并总结相关信息
- 创建子 Agent 并行处理独立任务
- 按自然语言计划运行定时任务
- 迁移 OpenClaw 的设置、记忆与技能

核心亮点：会积累经验的跨平台 Agent。学习循环把复杂任务沉淀为技能，并在使用中改进；记忆、会话搜索、定时调度、消息网关和并行子 Agent 共同支撑跨会话工作。

适合谁：需要让 Agent 在云端持续工作、跨会话记忆信息，或从聊天平台远程下达任务的用户与开发者
使用前准备：安装完成后需运行设置向导，并选择模型提供方与模型；部分工具可使用自备密钥或 Nous Portal。

使用提醒：Termux 的完整依赖包含与 Android 不兼容的语音组件，因此需使用专门的 Termux 安装项。

许可证：MIT

为什么本期上榜：过去 24 小时净增 +619 Star（历史快照核验）
GitHub 搜索：NousResearch/hermes-agent

03｜exercises-dataset
把 1,324 个健身动作整理成带动画、肌群数据和 10 种语言步骤的应用数据层

它能做什么：

- 读取 1,324 条结构化动作记录
- 筛选动作类别、器械与目标肌肉
- 查看 10 种语言的分步动作说明
- 用 JSON Schema 校验动作数据
- 生成多种数据库的建表与导入 SQL

核心亮点：可直接接入应用的动作数据层。JSON 数据把动作名称、器械、目标肌肉、多语言步骤和媒体路径放进统一结构；配套浏览器用于检索，设置页帮助导入数据库和连接后端 API。

适合谁：开发健身或训练计划应用、制作动作推荐模型，或需要多语言动作资料的开发者与研究者
使用前准备：浏览动作无需服务器，直接打开 `index.html`；接入应用时需将数据导入后端或直接读取 JSON。

使用提醒：代码、数据结构和说明文字使用 MIT；图片与 GIF 另受 Gym visual 条款约束，复用前需另行取得许可。
