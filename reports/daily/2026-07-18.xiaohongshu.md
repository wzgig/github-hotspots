01｜OpenCut
正在重写的开源视频编辑器，计划用一套代码覆盖网页、桌面和移动端

它能做什么：

- 计划开放 Editor API
- 计划接入第三方插件
- 计划让 AI 智能体通过 MCP 调用
- 计划执行自动化与批量渲染
- 计划在编辑器内运行脚本

核心亮点：一套核心覆盖多端与自动化。新版围绕 Rust 核心、插件优先架构和 Editor API 重建，并计划加入 MCP、无界面渲染与脚本入口，让交互剪辑和自动化处理共用同一套能力。

适合谁：适合跟进视频编辑器架构、插件系统和跨端方案的开发者；当前不适合直接依赖新版完成日常剪辑。
使用前准备：开发前需安装 proto，在仓库根目录运行 proto use，再用 moon 启动网页、API 或桌面端开发任务。

使用提醒：项目正在从头重写；当前应使用 classic 版本。架构设计期间尚不接受外部贡献。

许可证：MIT

02｜build-your-own-x
按技术主题汇集从零重造教程，帮助学习者通过亲手实现理解系统原理

它能做什么：

- 查找从零实现数据库的教程
- 学习编写操作系统与网络栈
- 练习构建编程语言和正则引擎
- 选择 AI 模型与神经网络指南
- 探索浏览器、服务器和 Git 的实现

核心亮点：用重造技术代替只读原理。它把分散的分步教程按技术类型整理，并标出教程使用的编程语言。读者可选定数据库、操作系统、AI 模型等目标，沿具体实现过程理解内部机制。

适合谁：适合希望通过重写现有技术理解内部机制的程序员，可按目标系统和熟悉的编程语言选择教程。

许可证：CC0
