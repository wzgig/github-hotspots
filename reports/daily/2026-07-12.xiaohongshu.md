01｜Catch2
Catch2 是面向 C++ 的单元测试框架，也提供基础微基准测试和简易 BDD 宏

它能做什么：

- 编写并运行 C++ 单元测试
- 使用自然语言为测试用例命名
- 通过断言核对表达式结果
- 复用局部准备与清理代码
- 显式运行带标签的微基准测试

核心亮点：贴近 C++ 表达式的测试写法。测试名不必是合法标识符，断言直接采用普通 C++ 布尔表达式；分区还能在局部共享准备和清理代码，让测试结构更自然。

适合谁：需要为 C++ 库或应用编写单元测试、组织 BDD 用例或测量局部代码耗时的开发者
使用前准备：需要用于 C++ 项目；当前版本按普通库方式使用，包含多个头文件和单独编译的实现。

使用提醒：微基准测试默认不会运行，需要使用 `[!benchmark]` 标签显式执行；版本文档仍在持续更新。

许可证：BSL-1.0

02｜firecrawl
firecrawl 通过 API 搜索、抓取和操作网页，并输出 Markdown、截图或结构化数据

它能做什么：

- 搜索网页并提取结果全文
- 把单个页面转成多种数据格式
- 遍历网站并抓取各页面内容
- 发现网站内的全部链接
- 按提示操作页面后再提取内容

核心亮点：从网页发现到交互式提取。同一套接口覆盖搜索、单页抓取、整站遍历、链接发现和批量任务；还可先点击、滚动或输入，再提取页面内容。

适合谁：需要为 AI 智能体、搜索应用或数据管道获取网页正文和结构化数据的开发者
使用前准备：使用托管服务需注册并获取 API Key；可通过 Python、Node.js、cURL 或 CLI 调用。

使用提醒：云端版本包含额外功能；抓取者需自行遵守网站政策、隐私政策和使用条款，默认遵循 robots.txt。

许可证：AGPL-3.0（The SDKs and some UI components are licensed under the MIT License.）

为什么本期上榜：过去 24 小时净增 +235 Star（历史快照核验）

03｜core
core 是 Home Assistant 的家庭自动化核心，强调在本地控制设备并保护隐私

它能做什么：

- 在本地运行家庭自动化系统
- 连接并控制受支持的家庭设备
- 通过模块扩展设备支持
- 添加新的自动化动作
- 开发自定义组件接入设备

核心亮点：模块化扩展家庭设备支持。系统采用模块化结构，设备与动作支持由组件接入；开发者可创建自己的组件，为新的设备类型或操作补充控制能力。

适合谁：希望在树莓派或本地服务器上管理家庭设备，以及开发自定义设备组件的用户与开发者
使用前准备：适合运行在 Raspberry Pi 或本地服务器上；安装步骤需参照项目文档。

许可证：Apache-2.0
