01｜hermes-agent
hermes-agent 把终端、聊天平台和云端环境连成一个能积累经验、持续执行任务的个人智能体

它能做什么：

- 切换不同模型提供方而无需改代码
- 连接多种聊天平台延续同一会话
- 创建定时任务并把结果发送到平台
- 派生隔离子智能体并行处理工作
- 迁移 OpenClaw 的设置、记忆和技能

核心亮点：经验驱动的学习闭环。它会从复杂任务中创建技能，在使用过程中继续改进，并结合会话搜索、长期记忆和用户模型，把一次性经验沉淀为后续可调用的工作方式。

适合谁：希望从终端或聊天软件管理长期任务，并让智能体逐步积累个人工作习惯的用户与开发者
使用前准备：安装后需选择模型提供方并完成配置；云端工具可使用自备密钥或 Nous Portal。

使用提醒：工具启用范围需自行配置；Windows 杀毒软件可能误报捆绑的 uv.exe，README 提供校验与白名单步骤。

许可证：MIT

为什么本期上榜：过去 24 小时净增 +478 Star（历史快照核验）
GitHub 搜索：NousResearch/hermes-agent

02｜awesome-llm-apps
从可运行模板起步，直接改造 AI Agent、RAG、语音交互和多智能体应用，不必反复重搭基础流程

它能做什么：

- 克隆模板搭建入门或进阶智能体
- 组合多个智能体处理跨领域任务
- 接入外部工具和数据构建 MCP 智能体
- 构建语音输入与语音输出应用
- 搭建多来源和多模态 RAG 流程

核心亮点：按任务挑选可运行模板。仓库按智能体、RAG、语音、多智能体、记忆和微调等场景组织独立项目，每个模板附源码，读者可先运行示例，再替换模型、数据和业务逻辑。

适合谁：需要快速验证 LLM 应用想法、学习常见实现方式，或寻找可改造项目骨架的开发者
使用前准备：需克隆仓库并按所选模板安装 requirements.txt；入门智能体通常还需模型 API Key。

使用提醒：不同模板使用的模型、框架和依赖不同，需进入对应目录核对配置；它不是一个统一部署的单体应用。

许可证：Apache-2.0

为什么本期上榜：过去 24 小时净增 +710 Star（历史快照核验）
GitHub 搜索：Shubhamsaboo/awesome-llm-apps

03｜Vibe-Trading
把自然语言里的交易问题接到市场数据、策略代码、回测和报告，形成可复查的量化研究流程

它能做什么：

- 输入交易问题生成可运行分析
- 回测策略并输出指标与验证材料
- 解析交易流水诊断行为偏差
- 比较实际交易与规则化影子策略
- 导出报告和多平台策略文件

核心亮点：从交易流水生成影子策略。上传券商导出记录后，它会统计交易行为、提取反复出现的进出场规则，再回测规则化的影子账户，对比违规、早退、漏信号和替代交易路径。

适合谁：需要研究市场、验证策略、复盘个人交易，或把量化分析能力接入现有智能体的研究者与开发者
使用前准备：需准备 LLM API Key 或本地 Ollama；本地安装要求 Python 3.11+，也可选择 Docker。

使用提醒：定位于研究、模拟和回测；自动交易仅在用户授权券商与设定限额内运行，部分连接器只读或仅支持模拟盘。

许可证：MIT
