# 正文

2026-07-12 · AI 专题榜

AI 项目的名字往往很抽象。这一期不堆术语，只讲清它替谁做什么、能完成哪些任务，以及是否值得加入你的工具箱。

先看结论：
- superpowers：superpowers 可以给编程 Agent 提供可复用的软件开发技能，同时让需求、编码和验证按固定流程推进
- firecrawl：想要搜索、抓取并整理网页内容，可以用 firecrawl；它能把网页数据交给程序或 AI 工作流继续处理
- hermes-agent：主要任务是搭建能够连续执行多步任务的 AI Agent；hermes-agent 还能把模型能力接入可重复的任务流程

01｜superpowers
superpowers 可以给编程 Agent 提供可复用的软件开发技能，同时让需求、编码和验证按固定流程推进

它能做什么：
- 把常用开发方法封装成可复用技能
- 按规划、实现和验证步骤组织任务
- 在不同项目中复用同一套开发流程

适合谁：希望让编程 Agent 按固定方法完成研发任务的开发者
为什么本期上榜：过去 24 小时净增 +277 Star（历史快照核验）
GitHub 搜索：obra/superpowers

02｜firecrawl
想要搜索、抓取并整理网页内容，可以用 firecrawl；它能把网页数据交给程序或 AI 工作流继续处理

它能做什么：
- 通过接口搜索并抓取公开网页
- 从页面中提取正文和结构化内容
- 把网页整理成更易处理的文本数据

适合谁：需要为 AI 应用、搜索或数据流程接入网页内容的开发者
为什么本期上榜：过去 24 小时净增 +235 Star（历史快照核验）
GitHub 搜索：firecrawl/firecrawl

03｜hermes-agent
主要任务是搭建能够连续执行多步任务的 AI Agent；hermes-agent 还能把模型能力接入可重复的任务流程

它能做什么：
- 把复杂任务拆成连续的执行步骤
- 让 Agent 围绕目标推进任务流程
- 为工具调用和工作流扩展提供基础

适合谁：需要搭建自动化智能体或多步骤 AI 工作流的开发者
为什么本期上榜：过去 24 小时净增 +232 Star（历史快照核验）
GitHub 搜索：NousResearch/hermes-agent
