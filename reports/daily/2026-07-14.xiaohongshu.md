01｜Vibe-Trading
把金融问题变成可运行的研究、策略回测和报告，并保留过程与结果供后续复用。

它能做什么：

- 查询市场数据并整理研究结论
- 生成策略代码并运行跨市场回测
- 分析交易日志中的行为偏差
- 调用多智能体团队并行完成投研
- 导出报告和多平台策略文件

核心亮点：从问题到可检查的研究产物。工作流先加载市场、文档或网页信息，再生成并执行策略，补充指标、验证和警告，最终交付报告、运行记录与可继续使用的导出文件。

适合谁：适合需要研究市场、验证策略、复盘交易记录，或把金融分析接入智能体工作流的个人与开发者。
使用前准备：需要可用的 LLM API Key，或本地运行 Ollama；本地安装要求 Python 3.11+，Docker 路径需要 Docker。

使用提醒：定位于研究、模拟与回测；实盘仅在用户授权和限额内运行，部分券商连接器只读或仅支持模拟盘。

许可证：MIT

02｜hermes-agent
让个人智能体从任务经验中沉淀记忆和技能，并跨终端与消息平台持续工作。

它能做什么：

- 保存记忆并检索过往会话
- 从任务经验创建和改进技能
- 安排定时任务并推送执行结果
- 派生隔离子智能体并行处理任务
- 切换模型和运行环境而不改代码

核心亮点：会积累经验的个人智能体。它把记忆、会话搜索、用户模型和技能系统连成学习闭环：复杂任务可沉淀为技能，后续使用时继续调整，并在不同会话中调用已有经验。

适合谁：适合希望长期使用个人智能体，自动处理定时任务、跨平台对话或并行工作流的用户与开发者。
使用前准备：安装器会准备 Python 3.11、Node.js、ripgrep 和 ffmpeg；使用前还需选择可用的模型提供方。

使用提醒：云端、消息平台和部分工具需要额外配置；Termux 的完整依赖包含当前不兼容的语音组件。

许可证：MIT

为什么本期上榜：过去 24 小时净增 +421 Star（历史快照核验）
GitHub 搜索：NousResearch/hermes-agent

03｜firecrawl
输入网址或数据需求，搜索并抓取网页，再输出适合程序和智能体处理的内容。

它能做什么：

- 搜索网页并返回结果全文
- 抓取页面并输出多种内容格式
- 发现站点内可访问的网址
- 批量异步处理多个页面
- 按提示操作页面后再提取内容

核心亮点：网页操作与内容提取合流。同一套接口覆盖搜索、单页抓取、整站遍历和页面交互；结果可整理成 Markdown、结构化数据或截图，便于直接接入程序与智能体流程。

适合谁：适合为智能体补充网页上下文、建设搜索与知识库，或批量采集网站内容的开发者。
使用前准备：使用托管接口需先注册并获取 API Key；也可以按照自托管文档运行开源版本。

使用提醒：抓取者须自行遵守网站政策、隐私政策和使用条款；默认遵循 robots.txt，云端版另含额外功能。

许可证：AGPL-3.0（The SDKs and some UI components are licensed under the MIT License.）
